# TKTools
Includes the tools I use in my projects


## Features
- Log (spdlog/stdout)
- Format (ANSI color codes/formatting)
- TKString (std::string wrapper)

## Requires
- xmake
- C++23

## Build
- `xmake project vsxmake -k` in repo root
