# TKTools
Some random tools I've made


## Features
- Log (spdlog/stdout)
  - Format (ANSI color codes/formatting)
- Dashboard (QT)
    - Components
- Events
- Handler
- Basic DLL Injector

## Requires
- [xmake](https://xmake.io/)
- C++23

## Build
- `xmake project vsxmake -k` in repo root
